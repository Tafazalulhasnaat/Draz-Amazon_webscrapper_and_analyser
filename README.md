Scrap & Analytics 

A Streamlit-based web app that scrapes product prices from Amazon & Daraz,
stores them in Firestore, and provides analytics and price history charts.

## Features
- Live scraping
- Price history tracking
- Mobile-friendly UI
- Firestore backend
- Streamlit frontend

## Tech Stack
- Python
- Streamlit
- Playwright
- Firebase Firestore
- Altair
